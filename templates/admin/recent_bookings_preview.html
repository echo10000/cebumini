Implementation

To include this component in your dashboard or dedicated bookings page, use:

{% include 'admin/recent_bookings_table.html' %}

Ensure you pass the context variable in your view:

from authentication.models import Booking def admin_dashboard(request): context = { 'recent_bookings': Booking.objects.all().order_by('-created_at')[:5] } return render(request, 'admin/dashboard.html', context)

Booking Model Fields Required

- guest (ForeignKey to User) - room (ForeignKey to Room) - check_in (DateField) - check_out (DateField) - status (CharField: 'confirmed', 'pending', etc.) - total_amount (DecimalField) - created_at (DateTimeField)
